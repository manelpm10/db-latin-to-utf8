Translate latin1 encoding to UTF8
=================================

Modify the script.php file and replace the next params.

* \<DB_HOST\>
* \<DB_USER\>
* \<DB_PASS\>
* \<DB_SCHEMA\>
* \<FIELD_TO_TRANSLATE\>
* \<TABLE_TO_TRANSLATE\>
 
Shity but useful script to translate a the database encoding for a field in a safe way.
